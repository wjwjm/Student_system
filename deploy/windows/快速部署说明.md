# Windows 快速部署说明（本地测试）

## 1. 前置条件
- Windows 10/11
- Docker Desktop（已启动）
- 建议内存 >= 8GB

## 2. 启动步骤
1. 解压 `student-system-windows.zip`。
2. 双击 `start.bat`。
3. 等待容器启动完成后，打开：
   - 系统主页：`http://localhost:8080/student/`
   - Swagger：`http://localhost:8080/student/swagger-ui.html`

默认账号：`admin / admin123`

## 3. 停止服务
双击 `stop.bat`。

## 4. 常见问题
- 8080/3306 端口占用：请释放端口后重试。
- Docker 未启动：先启动 Docker Desktop，再执行 `start.bat`。
